# Formulário Detran

[Para acessar o formulário, clique aqui:] (https://codepen.io/uchoamarcos/pen/QwGVMyZ)

## Cadastro de Clientes

Este projeto consiste em um formulário HTML responsivo para **cadastro de clientes**, utilizando a biblioteca [Bootstrap 5] (https://getbootstrap.com/) para estilização e organização dos elementos.

## Ferramenta utilizadas
```
- HTML 5
- [Bootstrap 5](https://getbootstrap.com/)
```

## Interface

O formulário inclui os seguintes campos:
```
- Nome
- Nome Social
- Doxumento de Identidade
- CPF e Órgão expedidor
- Nacionalidade
- Telefone
- Email
```
Como uma das ferramentas utilizadas fora o Bootstrap 5, os campos para o preenchimento de seus dados está no layout grid system do Bootstrap.
